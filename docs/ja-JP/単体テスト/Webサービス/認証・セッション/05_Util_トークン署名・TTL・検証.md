# 05. Util トークン署名・TTL・検証

## UT-05-001: 署名検証成功（正しい鍵・アルゴリズム）
- Given: 正しい `kid`/鍵束
- Then: AT/RT の署名検証 OK

## UT-05-002: 署名検証失敗（鍵不一致）
- Then: 401

## UT-05-003: TTL 満了で無効
- Given: `exp` を過去に
- Then: 検証で 401（`type=.../token_expired`）

## UT-05-004: token_version 埋め込み
- Given: `tv=10` で発行
- When: `logout_all` 実行（`tv=11` に更新）
- Then: 旧トークン検証時に **不一致で 401**
