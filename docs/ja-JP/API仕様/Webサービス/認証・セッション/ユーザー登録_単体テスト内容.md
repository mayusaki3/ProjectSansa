# テスト仕様：ユーザー登録（前段メール）

## 環境
- GitHub Actions / Linux
- Cassandra 単一 or 3ノード（LOCAL_QUORUM）
- MailHog（SMTP 1025, Web 8025）
- 固定クロック

## 観点
- 入力検証 / 重複 / ドメインブロック
- コード発行：最新のみ有効、TTL、再送クールダウン
- `preRegId`：1回限り、TTL、他端末無効（任意）
- 列挙対策：外形200
- i18n：`Accept-Language` と `Content-Language`

## 成果物
- 主要ケースのメールスナップショットをCIアーティファクト保存
