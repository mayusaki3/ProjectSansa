# ユーザー登録 API

## 目的
新規ユーザーを作成する。将来のメール確認/MFA を見据えた最小契約。

## エンドポイント
- **POST** `/auth/register`
- **認証**: 不要

## リクエスト
```http
POST /auth/register
Content-Type: application/json

{ "username":"<str>", "password":"<str>" }        // P1
{ "username":"<str>", "password":"<str>", "email":"<mail>" }  // P2+
