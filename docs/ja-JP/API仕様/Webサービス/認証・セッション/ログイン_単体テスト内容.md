# テスト仕様：ログイン（WebAuthn + MFA）

## 環境
- API 2台（api-1, api-2）
- Cassandra 1/3ノード、（任意）Redis Pub/Sub
- 固定クロック、MailHog

## 観点
- 正常：直近MFA済み端末→MFA不要
- `MFA_MAX_AGE` 超過→ `mfa_required` → TOTP/メールOTP成功
- 新規端末/高リスク時の `factors` 出し分け
- `logout_all` による 401、RT再利用検知での `tv++`
- i18n応答、レート制限
