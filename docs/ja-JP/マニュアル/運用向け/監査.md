# 監査（要点）

- 対象イベント：login/mfa/logout/refresh/email_verify/webauthn.register 等
- データ最小化：IPは粗化 or ハッシュ、UAはハッシュ
- 改ざん耐性：ハッシュチェーン + HMAC（kidローテ）
- API：`GET /me/security/activity`、`GET /admin/audit/search`

詳細は設計書（キャンバス版）を参照。
