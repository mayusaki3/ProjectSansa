# 初回登録ガイド

1. メールアドレスを入力して送信します。
2. 届いた6桁コードを入力します（またはメールのリンクをクリック）。
3. アカウントIDと言語を設定します。
4. 端末でパスキー（指紋/顔/端末PIN等）を登録します。VRではストロークで端末内鍵を解錠して利用します。

## よくある質問
- メールが届かない: 迷惑メールをご確認ください。1分後に再送できます。
- 端末を失くした: 「全端末ログアウト」を行い、別端末でパスキーを再登録してください。
