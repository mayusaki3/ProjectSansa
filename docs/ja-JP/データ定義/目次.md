# データ定義 目次

このディレクトリは **Webサービス（認証・セッション）**を中心とした**Cassandra スキーマ定義**と、
アプリ側の**論理データモデル**をまとめます。

- 物理設計は **単一ノード/3ノード**（RF=1/3）両対応
- Partition/Clustering Key、TTL、二次インデックス/補助ビューの方針を明記
- 監査・レート制限・ブロックリスト等の**運用系テーブル**も含める
